# AppQuizbiblicoMobile
Aplicativo desenvolvido em Flutter seguindo os princípios de Clean Architecture e Feature-First.

## 🛠 Tecnologias e Versões
* **Flutter SDK:** ^3.x.x
* **Gerenciamento de Estado:** flutter_bloc (Cubit)
* **Injeção de Dependências:** get_it (via contrato Injector)
* **Roteamento:** go_router
* **Equivalência de Objetos:** equatable
